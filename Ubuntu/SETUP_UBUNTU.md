# Ubuntu Installation

in terminal, go to the folder where you downloaded the script StripAndZipUnityProject.sh

and then:

```
cp StripAndZipUnityProject.sh to ~/.local/share/nautilus/scripts/StripAndZipUnityProject.sh
chmod +x ~/.local/share/nautilus/scripts/StripAndZipUnityProject.sh
```

The script is now available in the Nautilus file manager.
