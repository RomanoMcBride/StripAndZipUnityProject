Adds a context menu action to the file browser that zips Unity projects without generated folders like /Library (which often account for most of the project size).

https://github.com/user-attachments/assets/37d42549-0956-4271-a238-0f50ce3fe47b


# Usage

The project contains different scripts, one for each platform.

- [Mac](/Mac/SETUP_MAC.md)
- [Windows](/Windows/SETUP_WIN.md)
- Linux: the .sh in the mac folder can be run from the command line, provided that zip is installed.
